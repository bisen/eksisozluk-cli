### Build
```go
go build
```

### Run

Başlık ile arama:
```go
eksisozluk-cli baslik "golang"
```

Gündem Başlıkları:
```go
eksisozluk-cli gundem
```

DEBE:
```go
eksisozluk-cli debe
```
